# Release & Distribution Polish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Turn Cairn from a local-only `make run` app into a shippable macOS app: notarized DMG, Sparkle auto-update, README + usage docs, GitHub Actions release workflow. After this plan lands, a new tag on `main` cuts a notarized DMG that a user can download, drag to `/Applications`, and get auto-updates from.

**Architecture:** Release builds go through `scripts/release.sh` which chains existing `make swift` with a Release configuration, post-build codesigning (Developer ID Application), hardened runtime + the existing entitlements, `xcrun notarytool` submission using an `.env`-driven API key, and `scripts/make-dmg.sh` for the DMG. Sparkle is added via Swift Package Manager, wired through an `UpdaterService` wrapper exposed as a menu item. An `appcast.xml` is generated by `scripts/sparkle-appcast.sh` and published alongside the DMG on GitHub Releases. CI runs on tag pushes matching `v*.*.*` and uploads both.

**Tech Stack:** macOS codesign + notarytool, Sparkle 2.x (SPM), GitHub Actions (`macos-14` runner), Swift 5.9, existing `make-dmg.sh`.

**Scope boundary:** No Mac App Store in v1 (different signing, different review). No pkg installer. No delta updates (Sparkle full-DMG download). No in-app release notes renderer — the appcast links to a web-hosted changelog.

## Task 2: Notarization script

**Files:**
- Create: `scripts/notarize.sh`

- [ ] **Step 1: Shell script**

```bash
#!/usr/bin/env bash
# Submit an app bundle to Apple notarization and staple the ticket on success.
# Requires environment:
#   NOTARY_KEY_ID    — App Store Connect API key ID
#   NOTARY_ISSUER_ID — App Store Connect issuer UUID
#   NOTARY_KEY_PATH  — path to the .p8 key file
set -euo pipefail

APP="${1:-}"
[ -d "$APP" ] || { echo "usage: $0 <path/to/Cairn.app>" >&2; exit 1; }
: "${NOTARY_KEY_ID?NOTARY_KEY_ID not set}"
: "${NOTARY_ISSUER_ID?NOTARY_ISSUER_ID not set}"
: "${NOTARY_KEY_PATH?NOTARY_KEY_PATH not set}"

ZIP="$(mktemp -d)/Cairn.zip"
ditto -c -k --keepParent "$APP" "$ZIP"

echo "▶ Submitting to notarytool…"
xcrun notarytool submit "$ZIP" \
    --key "$NOTARY_KEY_PATH" \
    --key-id "$NOTARY_KEY_ID" \
    --issuer "$NOTARY_ISSUER_ID" \
    --wait

echo "▶ Stapling ticket…"
xcrun stapler staple "$APP"

echo "▶ Gatekeeper assessment:"
spctl --assess --type exec --verbose=4 "$APP"

echo "✓ Notarized: $APP"
```

```bash
#!/usr/bin/env bash
set -euo pipefail

VERSION="${1:-}"
[ -n "$VERSION" ] || { echo "usage: $0 <version, e.g. 1.0.0>" >&2; exit 1; }

: "${DEV_IDENTITY?DEV_IDENTITY must be set to sign the release}"

ROOT="$(cd "$(dirname "$0")/.." && pwd)"
cd "$ROOT"

echo "▶ Cleaning derived data"
rm -rf build/DerivedData

echo "▶ Building Release ($VERSION)"
CFBundleShortVersionString="$VERSION" make swift-release

APP="build/DerivedData/Build/Products/Release/Cairn.app"

echo "▶ Notarizing"
scripts/notarize.sh "$APP"

echo "▶ Building DMG"
scripts/make-dmg.sh "$APP" "build/Cairn-$VERSION.dmg"

echo "▶ Signing DMG"
codesign --sign "$DEV_IDENTITY" --timestamp "build/Cairn-$VERSION.dmg"

echo "▶ Notarizing DMG"
scripts/notarize.sh "build/Cairn-$VERSION.dmg"  # notarytool accepts dmgs too

echo "✓ Done. Artifact: build/Cairn-$VERSION.dmg"
```

```swift
import Foundation
import Sparkle

final class UpdaterService: NSObject {
    static let feedURL = URL(string: "https://github.com/ongjin/cairn/releases/latest/download/appcast.xml")!

    private let controller: SPUStandardUpdaterController

    override init() {
        self.controller = SPUStandardUpdaterController(
            startingUpdater: true,
            updaterDelegate: nil,
            userDriverDelegate: nil
        )
        super.init()
        controller.updater.feedURL = Self.feedURL
        controller.updater.automaticallyChecksForUpdates = true
    }

    func checkForUpdates() {
        controller.updater.checkForUpdates()
    }
}
```

## Task 6: Appcast generator

**Files:**
- Create: `scripts/sparkle-appcast.sh`

- [ ] **Step 1: Script — accept (version, dmg path, release-notes URL) + emit appcast.xml**

```bash
#!/usr/bin/env bash
set -euo pipefail

VERSION="${1:-}" DMG="${2:-}" NOTES_URL="${3:-}"
[ -n "$VERSION" ] && [ -f "$DMG" ] && [ -n "$NOTES_URL" ] \
  || { echo "usage: $0 <version> <dmg-path> <release-notes-url>" >&2; exit 1; }

SIZE=$(stat -f%z "$DMG")
DMG_URL="https://github.com/ongjin/cairn/releases/download/v${VERSION}/$(basename "$DMG")"
PUB_DATE=$(date -u +"%a, %d %b %Y %H:%M:%S +0000")

# Sign the DMG with the Sparkle EdDSA key (required by Sparkle 2+).
# Requires sparkle_signing_key.pem at repo root (gitignored).
SIG=$(./bin/sign_update "$DMG" | head -1)

cat <<XML > appcast.xml
<?xml version="1.0" encoding="utf-8"?>
<rss version="2.0" xmlns:sparkle="http://www.andymatuschak.org/xml-namespaces/sparkle">
  <channel>
    <title>Cairn</title>
    <link>https://github.com/ongjin/cairn</link>
    <item>
      <title>Version $VERSION</title>
      <sparkle:releaseNotesLink>$NOTES_URL</sparkle:releaseNotesLink>
      <pubDate>$PUB_DATE</pubDate>
      <enclosure
        url="$DMG_URL"
        sparkle:version="$VERSION"
        sparkle:shortVersionString="$VERSION"
        length="$SIZE"
        type="application/octet-stream"
        $SIG />
    </item>
  </channel>
</rss>
XML

echo "✓ Wrote appcast.xml"
```

## Task 7: GitHub Actions release workflow

**Files:**
- Create: `.github/workflows/release.yml`

- [ ] **Step 1: Workflow**

```yaml
name: Release
on:
  push:
    tags: ["v*.*.*"]

jobs:
  release:
    runs-on: macos-14
    steps:
      - uses: actions/checkout@v4
      - uses: actions/cache@v4
        with:
          path: |
            ~/.cargo/registry
            target
          key: ${{ runner.os }}-cargo-${{ hashFiles('**/Cargo.lock') }}

      - name: Install signing cert
        env:
          APPLE_CERT_P12_BASE64: ${{ secrets.APPLE_CERT_P12_BASE64 }}
          APPLE_CERT_PASSWORD:   ${{ secrets.APPLE_CERT_PASSWORD }}
        run: |
          echo "$APPLE_CERT_P12_BASE64" | base64 -d > /tmp/cert.p12
          security create-keychain -p action build.keychain
          security default-keychain -s build.keychain
          security unlock-keychain -p action build.keychain
          security import /tmp/cert.p12 -k build.keychain -P "$APPLE_CERT_PASSWORD" -T /usr/bin/codesign
          security set-key-partition-list -S apple-tool:,apple: -s -k action build.keychain

      - name: Install Sparkle signing key
        env:
          SPARKLE_PRIVATE_KEY_BASE64: ${{ secrets.SPARKLE_PRIVATE_KEY_BASE64 }}
        run: |
          echo "$SPARKLE_PRIVATE_KEY_BASE64" | base64 -d > sparkle_signing_key.pem

      - name: Release
        env:
          DEV_IDENTITY:    ${{ secrets.DEV_IDENTITY_SHA1 }}
          NOTARY_KEY_ID:   ${{ secrets.NOTARY_KEY_ID }}
          NOTARY_ISSUER_ID:${{ secrets.NOTARY_ISSUER_ID }}
          NOTARY_KEY_PATH: /tmp/notary.p8
          NOTARY_KEY_P8_BASE64: ${{ secrets.NOTARY_KEY_P8_BASE64 }}
        run: |
          echo "$NOTARY_KEY_P8_BASE64" | base64 -d > /tmp/notary.p8
          VERSION="${GITHUB_REF_NAME#v}"
          make release VERSION="$VERSION"
          ./scripts/sparkle-appcast.sh "$VERSION" "build/Cairn-$VERSION.dmg" \
              "https://github.com/ongjin/cairn/releases/tag/v$VERSION"

      - name: Upload to GitHub Release
        uses: softprops/action-gh-release@v2
        with:
          files: |
            build/Cairn-*.dmg
            appcast.xml
```

```markdown
# Cairn

A fast, native macOS file manager that treats local disks and SSH hosts as peers. Built with SwiftUI + a Rust core (indexing, SFTP).

[![Latest release](https://img.shields.io/github/v/release/ongjin/cairn)](https://github.com/ongjin/cairn/releases)

## Install

Download the latest `.dmg` from [Releases](https://github.com/ongjin/cairn/releases) and drag Cairn to `/Applications`. Cairn auto-updates via Sparkle; no package manager required.

## What you get

- Dual-pane folder navigation with `~` collapsing, breadcrumb, and a Finder-parity sidebar.
- Real SSH tabs backed by the native `~/.ssh/config` — host aliases, ProxyCommand, IdentityFile, all of it.
- Drag-and-drop upload/download with progress tracking.
- Streaming subtree search, local and remote.
- `cairn://` URL scheme + `cairn` CLI + Finder Services menu.

## Build from source

```
git clone https://github.com/ongjin/cairn
cd cairn
make run
```

Requires Xcode 17, Rust 1.80+, xcodegen (`brew install xcodegen`). The first build takes ~2 min (Rust universal static lib).
```
