# Remote Subtree Search Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** In SSH tabs, let the user type into the search bar with "Subtree" scope active and get recursive filename matches streamed from the remote host, cancellable, capped, with the same UX as the existing local subtree search.

**Architecture:** A new async `walk` API on the `SftpHandle` Rust type performs a BFS descent with a depth limit, a result cap, a cancellation flag, and a callback the FFI layer bridges to a Swift closure. `cairn-ffi` exposes `ssh_sftp_walk_start`, `ssh_sftp_walk_cancel`, and a "next batch" poll that returns a bounded slice of matches — identical cadence to the existing local walker so `SearchModel` needs minimal branching. `SshFileSystemProvider` gains a `walk(root:pattern:cap:cancel:)` async stream method. `SearchModel` gains a `walkRemote` path alongside the existing local branch.

**Tech Stack:** Rust (`async-trait`/`tokio::sync::mpsc` already in `cairn-ssh` pool), `russh-sftp` for directory iteration, swift-bridge FFI (same pattern as existing upload/download), Swift 5.9 `AsyncStream`, XCTest.

**Scope boundary:** Filename substring match only in v1 — no glob, no content grep. Caller supplies the max depth (default 10) and result cap (default 10_000). No fuzzy matching, no regex. Hidden files respect the app's existing `showHidden` toggle.

- [ ] **Step 2: Implement `walk`**

```rust
use std::sync::atomic::{AtomicBool, Ordering};
use std::sync::Arc;

impl SftpHandle {
    pub async fn walk<F>(
        &self,
        root: &str,
        opts: WalkOptions,
        cancel: Arc<AtomicBool>,
        mut emit: F,
    ) -> Result<()>
    where
        F: FnMut(WalkMatch),
    {
        use std::collections::VecDeque;
        let mut queue: VecDeque<(String, u32)> = VecDeque::from([(root.to_string(), 0u32)]);
        let mut emitted: usize = 0;
        let pattern_lc = opts.pattern.to_lowercase();

        while let Some((dir, depth)) = queue.pop_front() {
            if cancel.load(Ordering::Relaxed) { return Ok(()); }
            if emitted >= opts.cap { return Ok(()); }
            let entries = match self.list(&dir).await {
                Ok(e) => e,
                Err(_) => continue,  // skip unreadable dirs
            };
            for entry in entries {
                if cancel.load(Ordering::Relaxed) { return Ok(()); }
                if emitted >= opts.cap { return Ok(()); }
                let is_hidden = entry.name.starts_with('.');
                if is_hidden && !opts.include_hidden { continue; }
                let full = if dir.ends_with('/') { format!("{dir}{}", entry.name) }
                           else { format!("{dir}/{}", entry.name) };

                let name_matches = pattern_lc.is_empty()
                    || entry.name.to_lowercase().contains(&pattern_lc);
                if name_matches {
                    emit(WalkMatch {
                        path: full.clone(),
                        name: entry.name.clone(),
                        size: entry.size as i64,
                        is_directory: entry.is_directory,
                        mtime: entry.mtime.map(|d| d.timestamp()),
                    });
                    emitted += 1;
                }
                if entry.is_directory && depth + 1 < opts.max_depth {
                    queue.push_back((full, depth + 1));
                }
            }
        }
        Ok(())
    }
}
```

## Task 3: FFI surface — start / poll / cancel

- [ ] **Step 2: Add `ssh_sftp_walk_start`**

Takes `(handle, root, pattern, max_depth, cap, include_hidden) -> WalkSessionBridge`. Stores a `Vec<WalkMatch>` buffer protected by a Mutex, an `AtomicBool` cancel, and a `JoinHandle` that runs the walk and drops matches into the buffer.

```rust
pub struct WalkSessionBridge {
    buffer: Arc<tokio::sync::Mutex<Vec<WalkMatch>>>,
    cancel: Arc<AtomicBool>,
    join: Option<JoinHandle<Result<(), Error>>>,
}

pub fn ssh_sftp_walk_start(
    handle: &SftpHandleBridge,
    root: String, pattern: String, max_depth: u32, cap: u32, include_hidden: bool,
) -> Arc<WalkSessionBridge> {
    let buffer: Arc<_> = Arc::default();
    let cancel = Arc::new(AtomicBool::new(false));
    let buf_task = buffer.clone();
    let cancel_task = cancel.clone();
    let sftp = handle.inner.clone();
    let join = tokio::spawn(async move {
        sftp.walk(&root, WalkOptions {
            pattern, max_depth, cap: cap as usize, include_hidden,
        }, cancel_task, |m| {
            let buf = buf_task.clone();
            futures::executor::block_on(async move { buf.lock().await.push(m); });
        }).await
    });
    Arc::new(WalkSessionBridge { buffer, cancel, join: Some(join) })
}
```

(If `futures::executor::block_on` inside a closure looks wrong — it is, the Tokio alternative is to make `emit` async and use `walk`'s callback as an async closure; do whichever matches the rest of the FFI. The key invariant is: matches end up in `buffer` without blocking the FFI caller.)

- [ ] **Step 3: Add drain / cancel functions**

```rust
pub fn ssh_sftp_walk_drain(session: &WalkSessionBridge) -> Vec<WalkMatch> {
    let mut buf = futures::executor::block_on(async { session.buffer.lock().await });
    std::mem::take(&mut *buf)
}

pub fn ssh_sftp_walk_cancel(session: &WalkSessionBridge) {
    session.cancel.store(true, Ordering::Relaxed);
}

pub fn ssh_sftp_walk_is_done(session: &WalkSessionBridge) -> bool {
    session.join.as_ref().map(|h| h.is_finished()).unwrap_or(true)
}
```

```swift
func walk(root: FSPath, pattern: String, maxDepth: Int, cap: Int,
          includeHidden: Bool, cancel: CancelToken) -> AsyncThrowingStream<FileEntry, Error> {
    AsyncThrowingStream { continuation in
        let session = ffi_ssh_sftp_walk_start(
            handle: self.handle,
            root: root.path, pattern: pattern,
            maxDepth: UInt32(maxDepth), cap: UInt32(cap),
            includeHidden: includeHidden
        )
        Task.detached {
            while !Task.isCancelled {
                if cancel.isCancelled {
                    ffi_ssh_sftp_walk_cancel(session: session)
                    break
                }
                let drained = ffi_ssh_sftp_walk_drain(session: session)
                for m in drained {
                    let entry = FileEntry(
                        name: m.name,
                        path: FSPath(provider: .ssh(self.target), path: m.path),
                        size: m.size,
                        mtime: m.mtime.map { Date(timeIntervalSince1970: TimeInterval($0)) },
                        kind: m.isDirectory ? .Directory : .Regular,
                        mode: 0o644
                    )
                    continuation.yield(entry)
                }
                if ffi_ssh_sftp_walk_is_done(session: session) && drained.isEmpty {
                    break
                }
                try? await Task.sleep(nanoseconds: 100_000_000)  // 100ms poll
            }
            continuation.finish()
        }
    }
}
```

```swift
func refresh(
    root: FSPath?,
    provider: FileSystemProvider,
    showHidden: Bool,
    sort: FolderModel.SortDescriptor,
    folderEntries: [FileEntry]
) {
    task?.cancel()
    task = nil
    cancel?.cancel()
    if case .ssh = provider.identifier, scope == .subtree, !query.isEmpty {
        runRemoteSubtree(root: root, provider: provider, showHidden: showHidden)
    } else {
        // existing local/folder logic
    }
}

private var cancel: CancelToken?

private func runRemoteSubtree(root: FSPath?, provider: FileSystemProvider, showHidden: Bool) {
    guard let root else { phase = .idle; return }
    phase = .running
    results = []
    hitCount = 0
    let token = CancelToken()
    self.cancel = token
    self.task = Task { @MainActor [weak self] in
        guard let self else { return }
        do {
            let stream = provider.walk(
                root: root, pattern: self.query, maxDepth: 10,
                cap: 10_000, includeHidden: showHidden, cancel: token
            )
            var next: [FileEntry] = []
            for try await entry in stream {
                if Task.isCancelled { break }
                next.append(entry)
                if next.count >= 200 {
                    self.results.append(contentsOf: next)
                    self.hitCount = self.results.count
                    next.removeAll(keepingCapacity: true)
                }
            }
            self.results.append(contentsOf: next)
            self.hitCount = self.results.count
            self.phase = .done
        } catch {
            self.phase = .failed(String(describing: error))
        }
    }
}
```
